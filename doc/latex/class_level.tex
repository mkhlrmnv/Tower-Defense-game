\doxysubsection{Detailed Description}
Class that controls level of the game. 

Class holds current grid, all enemies and tower. Also it keeps track of round, money and live situation \doxylink{class_level}{Level} can load hard coded maps from files, save current levels to files or randomly generate fully new one
